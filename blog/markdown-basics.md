Markdown is a lightweight markup language that's perfect for writing blog posts. Here are the basics:

## Headings

```
# Heading 1
## Heading 2
### Heading 3
```

## Emphasis

You can make text **bold** or *italic*.

## Lists

### Unordered Lists

- Item 1
- Item 2
- Item 3

### Ordered Lists

1. First item
2. Second item
3. Third item

## Links

[Visit Eleventy](https://11ty.dev/)

## Code

Inline `code` looks like this. For code blocks:

```javascript
console.log("Hello, Sakura!");
```

## Blockquotes

> "The greatest glory in living lies not in never falling, but in rising every time we fall." - Nelson Mandela

That's it! Markdown makes writing content fun and easy.
