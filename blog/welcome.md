Welcome to Sakura Blog! This is your first blog post. You can edit or delete this post by opening the blog folder.

## Why Eleventy?

Eleventy is a simpler static site generator that:
- Requires zero config by default
- Works with your favorite template language
- Outputs multiple file types
- Is lightweight and fast

## Why Sakura CSS?

Sakura is a minimal CSS framework that:
- Provides beautiful default styles
- Is just a few kilobytes
- Requires no classes or complex markup
- Works great for blogs and documentation

Start writing your own posts in the `blog/` directory!
